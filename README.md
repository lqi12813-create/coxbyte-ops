# Coxbyte 运营官

专为 Coxbyte 品牌定制的跨境电商运营专家，内置全球导热硅脂市场 29 个竞品完整画像，覆盖 Amazon/速卖通/Shopee 渠道优化、SEO 内容策略、竞品对标与定价分析。

## 核心能力

- **竞品监测**：内置 Arctic、Thermal Grizzly、利民、九州风神等 29 个品牌完整数据
- **Amazon 优化**：产品页面标题、五点描述、A+ 内容、关键词策略
- **SEO 内容规划**：全球关键词机会分析、内容日历、长尾词布局
- **定价策略**：基于每克价格对比的定价建议与甜点区定位
- **渠道诊断**：Amazon/AliExpress/淘宝/京东/Shopee 多平台表现评估

## 触发方式

在 WorkBuddy 中选择「Coxbyte运营官」专家，或发送以下类型的问题：

- "分析 Coxbyte 本周 Amazon 销量，对比 Arctic"
- "优化 TG-50 的 Amazon 产品页面"
- "制定下个月 Coxbyte SEO 内容日历"
- "Coxbyte 定价应该定多少？"
- "监测 Thermal Grizzly 最近的 Amazon 排名变化"

## 内置知识来源

基于 2026 年 5 月全球导热硅脂市场深度调研报告（34 个调研对象、40+ 维度），数据来源包括：
Tom's Hardware、Club386、Igor's Lab、Amazon Best Sellers、京东排行榜、中国品牌网等。
